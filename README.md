# Ergo Protocol

Scaffolded workspace for the Ergo Protocol monorepo.

This repository is organized as a multi-package codebase for:

- Soroban smart contracts under `contracts/`
- Shared Rust types under `shared/`
- Off-chain services under `server/` and `keepers/`
- The Next.js frontend under `client/`
- Deployment helpers under `scripts/`
- Integration coverage under `tests/integration/`

The full project architecture is described in the root `copilot-instructions` file and mirrored in `.github/copilot-instructions.md`.
