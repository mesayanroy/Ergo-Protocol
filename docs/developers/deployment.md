# 🚀 Deployment Guide

This document describes how to compile, optimize, and deploy Ergo Protocol smart contracts to Stellar.

## Deployed Networks

1. **Stellar Testnet**: Used for staging and integration testing.
2. **Stellar Mainnet**: Public deployment network.

## Commands

```bash
# Optimize WASM binaries
python scripts/optimize_wasm.py

# Run deploy scripts
node scripts/deploy-mainnet.js
```
