# 🏥 Backstop Pool & Protocol Insurance

The Backstop Pool is Ergo Protocol's primary defense layer against black swan insolvency events.

## Features

- **USDC Underwriting**: Liquidity providers deposit USDC to underwrite protocol shortfall events.
- **Token Rewards**: Backstop depositors earn ERGO protocol emissions in exchange for absorbing risk.
- **Shortfall Recovery**: If a liquidation auction fails to recover debt fully, the Backstop contract covers the remaining deficit.
